# 贰 · 语法基础

在 [**壹 · 引言**](../intro/README.md) 中，我们已经学习了如何安装 CovScript、设置开发环境，并编写了第一个 "Hello, World!" 程序。现在，让我们深入学习 CovScript 的语法和特性。

本章介绍 CovScript 的语法基础，包括基本语法规则、数据类型、运算符、控制流、函数、面向对象编程等核心内容。

CovScript 是一门灵活而强大的动态语言，语法简洁明了，容易上手。本章内容适合所有水平的开发者，从初学者到高级用户都能从中受益。

**注意：** 本手册主要基于 CovScript 4 (ECS) 进行展示，同时会标注与 CovScript 3 (CSC) 的区别。
